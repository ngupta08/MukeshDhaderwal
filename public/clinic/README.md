# Clinic Photos Directory

Add your clinic photos here with the following naming convention:

- `reception.jpg` - Reception area photo
- `consultation.jpg` - Consultation room photo
- `waiting.jpg` - Waiting area photo
- `exterior.jpg` - Clinic exterior/building photo
- `placeholder.jpg` - Fallback image (optional)

## Image Requirements:
- Recommended size: 1200x800px or larger
- Format: JPG, PNG, or WebP
- File size: Optimize images to keep file sizes reasonable (< 500KB per image)

## Alternative: Using Cloudinary URLs

You can also update the `clinicPhotos` array in `components/ClinicPhotos.tsx` to use Cloudinary URLs instead of local images by changing the `imageUrl` property to your Cloudinary image URLs.
